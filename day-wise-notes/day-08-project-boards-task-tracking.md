## ğŸ” What is Task Tracking?
Task tracking is the process of:
- Breaking a project into smaller tasks
- Assigning tasks to team members
- Tracking progress of each task
- Ensuring work is completed on time

Without task tracking, team projects become disorganized and confusing.

## ğŸ§© What are GitHub Projects?
GitHub Projects is a **visual task management tool** provided by GitHub.
It works like a kanban board (similar to Trello or Jira).

GitHub Projects help you:
- Track Issues and Pull Requests

- Manage collaboration in teams
- See the overall progress of a project
